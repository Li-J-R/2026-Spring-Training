# Linux 学习笔记



## 1. Linux 基础概念



### 1.1 什么是 Linux？

- Linux 是一个免费、开源、类 Unix 的操作系统内核。

- 基于 Linux 内核的操作系统称为发行版（如 Ubuntu、CentOS、Debian、Fedora）。



### 1.2 核心特性

- 多用户、多任务

- 稳定、安全、高效

- 一切皆文件（硬件、目录、设备都表示为文件）



### 1.3 常用发行版

| 发行版     | 包管理工具       | 适用场景           |

|------------|----------------|--------------------|

| Ubuntu     | apt             | 桌面、服务器、开发 |

| CentOS / RHEL | yum / dnf     | 企业服务器         |

| Debian      | apt             | 稳定服务器         |

| Fedora      | dnf             | 最新技术体验       |

| Arch Linux  | pacman          | 高度定制化         |

## 2. Linux 文件系统



### 2.1 目录结构（FHS 标准）

| 目录   | 说明                                 |

|--------|--------------------------------------|

| /      | 根目录                               |

| /bin   | 基础系统命令（二进制文件）           |

| /sbin  | 系统管理命令（需要 root 权限）       |

| /etc   | 配置文件                             |

| /home  | 普通用户的家目录                       |

| /root  | root 用户的家目录                      |

| /var   | 可变数据（日志、缓存、邮件）           |

| /tmp   | 临时文件（重启后可能清空）           |

| /usr   | 用户程序和数据（类似 Windows 的 Program Files） |

| /proc  | 虚拟文件系统，存储进程和内核信息       |



### 2.2 路径表示

- **绝对路径**：从根 `/` 开始，如 `/home/user/file.txt`

- **相对路径**：相对于当前目录，如 `./file.txt`、`../dir/`

## 3. 基础命令



### 3.1 文件和目录操作

| 命令          | 作用                     | 示例                         |

|---------------|--------------------------|------------------------------|

| `ls`          | 列出目录内容             | `ls -la`（显示所有文件详情） |

| `cd`          | 切换目录                 | `cd /etc`                    |

| `pwd`         | 显示当前路径             | `pwd`                        |

| `mkdir`       | 创建目录                 | `mkdir -p a/b/c`（递归创建） |

| `rmdir`       | 删除空目录               | `rmdir empty_dir`            |

| `rm`          | 删除文件或目录           | `rm -rf dir/`（强制递归删除）|

| `cp`          | 复制文件或目录           | `cp -r src/ dst/`            |

| `mv`          | 移动/重命名              | `mv old.txt new.txt`         |

| `touch`       | 创建空文件或更新时间戳   | `touch file.txt`             |

| `file`        | 查看文件类型             | `file script.sh`             |



### 3.2 查看文件内容

| 命令      | 作用                           | 示例                |

|-----------|--------------------------------|---------------------|

| `cat`     | 显示全部内容                   | `cat file.txt`      |

| `less`    | 分页浏览（空格翻页，q 退出）   | `less large.log`    |

| `head`    | 显示前10行                     | `head -n 20 file`   |

| `tail`    | 显示后10行                     | `tail -f log`（实时跟踪） |

| `grep`    | 搜索文本                       | `grep "error" log`  |



### 3.3 权限管理

- **权限表示**：`rwxr-xr--` 分为三组（所有者、组、其他）

  - `r` = 读（4）

  - `w` = 写（2）

  - `x` = 执行（1）



| 命令        | 作用                     | 示例                             |

|-------------|--------------------------|----------------------------------|

| `chmod`     | 修改权限（符号或数字）   | `chmod 755 script.sh`            |

| `chown`     | 修改所有者:组            | `chown user:group file`          |

| `chgrp`     | 修改所属组               | `chgrp staff file`               |



### 3.4 帮助文档

| 命令        | 作用                     | 示例                |

|-------------|--------------------------|---------------------|

| `man`       | 查看命令手册             | `man ls`            |

| `info`      | 更详细的文档             | `info coreutils`    |

| `--help`    | 简要帮助                 | `ls --help`         |

| `groups`    | 显示用户所属组           | `groups alice`               |



### 4.3 切换用户

|-------------|--------------------------|---------------------|

| `su`        | 切换用户（需目标用户密码）| `su - alice`        |

## 5. 进程管理



| 命令        | 作用                     | 示例                         |

|-------------|--------------------------|------------------------------|

| `nohup`     | 忽略挂断信号运行         | `nohup ./server &`           |

## 7. 网络配置与管理



| 命令            | 作用                               | 示例                         |

|-----------------|------------------------------------|------------------------------|

| `ip addr` / `ifconfig` | 查看IP地址（需安装net-tools） | `ip a`                     |

### 7.1 防火墙（以 ufw 为例）

```bash

sudo ufw allow 22/tcp          # 允许 SSH

sudo ufw allow 80/tcp          # 允许 HTTP

sudo ufw status verbose        # 查看状态

```

## 11. 常用技巧与快捷键



### 11.1 命令行快捷键

| 快捷键       | 作用                     |

|--------------|--------------------------|

| `Tab`        | 自动补全命令/路径        |

| `Ctrl + C`   | 终止当前进程             |

| `Ctrl + Z`   | 暂停进程（放入后台）     |

| `Ctrl + D`   | 退出当前终端（发送 EOF） |

| `Ctrl + L`   | 清屏（类似 `clear`）    |

| `Ctrl + A`   | 光标移至行首             |

| `Ctrl + E`   | 光标移至行尾             |

| `↑` / `↓`    | 浏览命令历史             |

| `!!`         | 重复上一条命令           |

| `!n`         | 执行历史中第 n 条命令    |



### 11.2 重定向与管道

- **重定向**：

  - `>`  覆盖输出到文件

  - `>>` 追加输出到文件

  - `2>` 重定向错误输出

  - `&>` 重定向所有输出

- **管道 `|`**：将前一个命令的输出作为后一个命令的输入

  ```bash

  ls -la | grep ".txt" | wc -l

  ```

## 13. 常见问题排查思路



| 问题类型       | 常用命令/方法                              |

|----------------|--------------------------------------------|

| 磁盘满         | `df -h` 查看使用率；`du -sh /*` 找大文件 |

| CPU 高负载     | `top` / `htop` 看进程；`ps aux --sort=-%cpu` |

| 内存不足       | `free -h`；`ps aux --sort=-%mem`         |

| 端口被占用     | `ss -tuln` 或 `netstat -tuln`；`lsof -i:端口号` |

| 无法上网       | `ping 8.8.8.8`；检查 DNS `cat /etc/resolv.conf`；`ip a` 看IP配置 |

| 权限被拒绝     | 检查文件权限 `ls -l`；使用 `sudo`；查看用户组 `id` |
